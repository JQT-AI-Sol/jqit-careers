職種紹介・募集要項

あなたの経験と、挑戦したい領域で選べる4職種。チームを率いて活かせるポジションです。

Engineering

開発エンジニア

経験者歓迎

Androidアプリ・iOSアプリの開発を中心に、要件検討から設計・実装・リリース後の改善まで担当します。大手キャリア向けアプリなど、ユーザーに近いモバイルプロダクトづくりに携わります。

求める人物像

  • ものづくりが好きで、技術を学び続けられる方
  • チームで価値を届けることに喜びを感じる方
  • アプリ／Web開発の実務経験を、チームを率いて活かしたい方

歓迎するスキル・経験

  • 何らかの言語でのプログラミング経験
  • Androidアプリ開発（Kotlin / Java等）の経験
  • iOSアプリ開発（Swift等）の経験

プロジェクト事例

  • 大手キャリア向けAndroidアプリ開発（要件検討〜仕様調整）
  • 大手キャリア向けiOSアプリ開発（設計・実装・テスト）
  • 既存スマートフォンアプリの機能追加・改善対応
  • モバイルアプリと連携するAPI仕様調整・動作確認
この職種に応募 / 相談する
Infrastructure

インフラエンジニア

経験者歓迎

クラウドを中心としたインフラの設計・構築・運用を担います。IaCによる自動化、監視・セキュリティ設計まで、安定とスケールを支える基盤づくりに取り組みます。

求める人物像

  • 縁の下で仕組みを支えることにやりがいを感じる方
  • 自動化・効率化を追求できる方
  • クラウド／インフラの実務経験を、IaC・自動化で次の段階へ進めたい方

歓迎するスキル・経験

  • Linux/ネットワークの基礎知識
  • AWS/GCP/Azure いずれかの実務経験
  • IaC（Terraform等）・CI/CDの経験

プロジェクト事例

  • 企業基幹システムのネットワーク設計・構築（機器更改）
  • オンプレ仮想環境からクラウドIaaSへの移行（Apache / Java EE / PostgreSQL）
  • 某クレジット会社の基盤構築（AWS構築・Linux設計・ミドルウェア設計）
  • パブリッククラウド接続ネットワーク提供サービスの方式設計
この職種に応募 / 相談する
Quality Assurance

QAエンジニア

経験者歓迎

テスト設計・実行・自動化、品質プロセスの構築までを担当します。「使う人」を想う日本品質を、技術で体現する重要なポジションです。社内のエンジニアで最も多いのがQAという、品質に強みを持つ会社です。

求める人物像

  • 品質にこだわり、細部に目を配れる方
  • プロダクトを良くする提案を楽しめる方
  • テスト／QAの実務経験を、自動化・品質プロセス構築まで広げたい方

歓迎するスキル・経験

  • テスト設計・実行の経験
  • テスト自動化（Playwright/Selenium等）の経験
  • JSTQB / IVEC 等の資格・知識

プロジェクト事例

  • スマートフォン向けゲームアプリの品質検証
  • 宅配サービスWebシステムのシステムテスト・テストケース作成
  • 決済アプリの品質検証（テスト設計〜実施）
  • ECサイトパッケージの品質管理（品質保証計画の立案・標準化）
この職種に応募 / 相談する
Artificial Intelligence

AIエンジニア

経験者歓迎

大規模言語モデルを中心とした最先端技術で、AIトランスフォーメーションを推進します。AIエージェントの開発や生成AIのビジネス活用など、新規事業の最前線に挑む、経験者向けのポジションです。

求める人物像

  • 最新のAI技術に強い関心と探究心がある方
  • 機械学習・生成AIの実装/PoC経験がある方
  • 自走して新しい技術を検証し、価値を生み出せる方

歓迎するスキル・経験

  • Python等での開発経験
  • LLM/生成AIを用いた開発・PoCの経験
  • 機械学習・データ処理の知識

プロジェクト事例

  • 大手通信キャリアの社内向けLLMアプリ開発
  • 生成AIを活用したシステムの品質・精度検証
  • AIエージェント・生成AIのビジネス活用支援（PoC〜導入）
この職種に応募 / 相談する
